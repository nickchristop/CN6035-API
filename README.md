# CN6035-API

Theatre seat reservation backend — Node.js, Express, MariaDB.

## Development History Note

Early backend development was initially completed without a proper feature-based
commit workflow. This repository's commit history from this point has been
retrospectively reconstructed so that the project's development stages are
clearer for review and evaluation. The reconstruction is intentionally labelled
as retrospective; from this point onward, development continues using regular
feature-based commits.
